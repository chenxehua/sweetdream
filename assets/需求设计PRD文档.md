## 1. 产品概述与目标用户画像

### 1.1 产品概述

#### 产品定位
针对3-12岁儿童的睡眠健康应用，通过睡前仪式引导、情绪疏导内容和家长育儿指导，帮助家庭建立科学的睡眠习惯，从根本上改善儿童睡眠问题。

#### 核心价值主张
- **对于儿童**：通过温暖的睡前仪式和情绪疏导内容，帮助孩子建立安全感、缓解焦虑，自然进入睡眠
- **对于家长**：提供科学的睡眠知识和操作指南，帮助家长理解睡眠原理、减少哄睡焦虑

#### 产品slogan（备选）
- "每晚一个睡眠好习惯"
- "让孩子睡得更安心"
- "科学的睡前仪式，温暖的睡眠陪伴"

### 1.2 目标用户画像

#### 主用户画像：焦虑的新手妈妈

| 属性 | 描述 |
|------|------|
| **基础信息** | 28-35岁，本科学历，一/二线城市 |
| **生活状态** | 双职工家庭，孩子3-6岁，老人帮忙照看 |

## 4. 用户故事与验收标准

### 4.1 核心用户故事

#### 故事1：忙碌的妈妈快速开始哄睡

| 项目 | 内容 |
|------|------|
| **作为** | 一位有两个孩子的职场妈妈 |
| **我想要** | 快速开始哄睡流程，不需要复杂操作 |
| **以便** | 在有限的时间内高效哄睡两个孩子 |
| **场景** | 晚上9点下班回家，9点半开始哄睡 |
| **验收标准** | 1. 打开App后3秒内可点击"开始哄睡"<br>2. 上次使用的仪式自动展示<br>3. 播放过程中一只手可完成所有操作 |

#### 故事2：焦虑的孩子找到安全感

| 项目 | 内容 |
|------|------|
| **作为** | 一位4岁、对黑暗害怕的孩子 |
| **我想要** | 有温暖的声音陪伴我入睡 |
| **以便** | 不再害怕黑暗，安心入睡 |
| **场景** | 关灯后害怕，叫妈妈陪 |
| **验收标准** | 1. 可选择喜欢的故事类型（动物、公主等）<br>2. 播放的声音温暖、舒缓<br>3. 音量适中，不会突然变化 |

#### 故事3：学习型妈妈获取专业指导

|------|------|
| **作为** | 一位关注科学育儿的妈妈 |
| **我想要** | 了解孩子睡眠问题的原因和解决方法 |
| **以便** | 更科学地帮助孩子改善睡眠 |
| **场景** | 孩子持续一个月睡眠不好，想找专业方案 |

#### 故事4：家长追踪孩子睡眠改善

|------|------|

### 4.2 功能验收标准

#### 验收标准模板

| 功能 | 验收条件 | 测试方法 |
|------|---------|---------|
| 功能名称 | 明确描述期望行为 | 描述用户操作和系统响应 |
| 前置条件 | 使用该功能前需满足的条件 | 列出登录状态、权限等 |
| 测试步骤 | 具体的操作步骤 | 1-2-3列出 |
| 预期结果 | 期望的系统行为 | 明确具体结果 |
| 通过标准 | 什么情况下判定通过 | 量化指标 |

#### 睡前仪式-验收标准

```
【功能名称】睡前仪式引导
【前置条件】已登录家长账号，孩子档案已创建
【测试步骤】
1. 打开App，进入睡前仪式页面
2. 选择一个仪式模板
3. 点击"开始仪式"
4. 跟随引导完成仪式
5. 仪式结束后进入睡眠

【预期结果】
- 仪式模板列表展示完整（≥5个模板）
- 每个模板显示名称、图标、预估时长
- 仪式开始后按步骤依次展示
- 每个步骤可自动推进或手动点击
- 仪式完成有明确提示

【通过标准】
- 仪式流程完整走通，无中断
- 每个步骤响应时间 < 1秒
- 仪式进度清晰可见
```

#### 音频播放-验收标准

```
【功能名称】音频内容播放
【前置条件】网络连接正常，音频已加载
【测试步骤】
1. 进入音频列表页
2. 点击任意音频开始播放
3. 操作播放控制（暂停/继续/切歌）
4. 设置定时关闭
5. 切换App到后台
6. 返回App

【预期结果】
- 音频列表加载时间 < 2秒
- 播放流畅，无卡顿
- 播放控制响应及时
- 定时关闭生效
- 后台播放正常

【通过标准】
- 连续播放30分钟无中断
- 内存占用稳定，无明显增长
- 耗电量控制在合理范围
```

## 6. 非功能需求

### 6.1 性能需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 首屏加载时间 | ≤2秒 | 首次打开时间 |
| 页面切换时间 | ≤1秒 | 页面间跳转 |
| 音频播放启动 | ≤1秒 | 点击播放到出声 |
| 离线可用性 | 核心功能支持 | 睡前仪式流程可离线 |
| 崩溃率 | ≤0.1% | 行业优秀水平 |
| ANR率 | ≤0.05% | 应用无响应 |

### 6.2 安全需求

| 需求 | 描述 | 实施措施 |
|------|------|---------|
| 账号安全 | 微信账号体系安全 | 微信登录机制保障 |
| 数据传输 | HTTPS加密传输 | 全站HTTPS |
| 数据存储 | 敏感数据加密存储 | 密码、支付信息加密 |
| 儿童数据 | 特殊保护 | 仅家长可见孩子数据 |
| 支付安全 | 微信支付安全 | 微信支付原生支持 |

### 6.3 隐私需求

| 需求 | 描述 | 实施措施 |
|------|------|---------|
| 知情同意 | 明确告知数据收集用途 | 隐私政策弹窗 |
| 家长授权 | 14岁以下需家长同意 | 家长账号验证 |
| 数据最小化 | 仅收集必要数据 | 功能导向设计 |
| 删除权 | 支持删除账号数据 | 设置中提供删除入口 |
| 第三方共享 | 明确告知第三方 | 隐私政策说明 |

### 6.4 兼容性需求

| 类型 | 要求 |
|------|------|
| iOS版本 | iOS 12.0+ |
| Android版本 | Android 8.0+ |
| 微信版本 | 微信7.0+ |
| 屏幕适配 | iPhone SE 到 iPhone 15 Pro Max |
| 平板适配 | iPad基础支持 |

### 6.5 辅助功能需求

| 需求 | 描述 |
|------|------|
| 屏幕朗读 | 关键功能支持屏幕朗读 |
| 操作简化 | 儿童端操作简单，无复杂手势 |
| 色彩对比 | 文字与背景对比度 ≥ 4.5:1 |
| 字体大小 | 支持系统字体大小设置 |

## 7. 数据埋点需求

### 7.1 埋点原则

| 原则 | 说明 |
|------|------|
| 必要性 | 只埋点对产品优化必要的数据 |
| 合法性 | 遵守隐私法规，明确告知用户 |
| 最小化 | 收集最少的数据满足分析需求 |
| 安全性 | 埋点数据传输加密存储 |

### 7.2 核心事件埋点

#### 7.2.1 用户行为事件

| 事件ID | 事件名称 | 触发时机 | 携带参数 |
|--------|---------|---------|---------|
| page_view | 页面浏览 | 进入页面 | 页面名称、来源 |
| button_click | 按钮点击 | 点击按钮 | 按钮ID、位置 |
| ritual_start | 仪式开始 | 点击开始仪式 | 仪式ID、仪式类型 |
| ritual_complete | 仪式完成 | 仪式执行完毕 | 仪式ID、时长、完成度 |
| audio_play | 音频播放 | 点击播放 | 音频ID、音频类型 |
| audio_finish | 音频播放完成 | 音频播完 | 音频ID、播放时长 |
| settings_save | 设置保存 | 保存设置 | 设置项、设置值 |

#### 7.2.2 睡眠数据事件

| 事件ID | 事件名称 | 触发时机 | 携带参数 |
|--------|---------|---------|---------|
| sleep_record | 睡眠记录 | 记录睡眠 | 入睡时间、起床时间、质量评分 |
| sleep_report_view | 报告查看 | 查看报告 | 报告类型、时间范围 |
| sleep_trend_view | 趋势查看 | 查看趋势 | 时间范围 |

#### 7.2.3 商业转化事件

| 事件ID | 事件名称 | 触发时机 | 携带参数 |
|--------|---------|---------|---------|
| vip_page_view | 会员页浏览 | 进入会员页 | 来源页面 |
| vip_try_click | 试用点击 | 点击试用 | 会员类型 |
| order_confirm | 订单确认 | 确认支付 | 订单金额、商品ID |
| pay_success | 支付成功 | 支付完成 | 订单金额、支付方式 |

### 7.3 页面埋点

| 页面 | 页面ID | 关键指标 |
|------|--------|---------|
| 首页 | home | 浏览量、停留时长 |
| 睡前仪式页 | ritual | 仪式启动率、完成率 |
| 音频播放页 | player | 播放完成率、收藏率 |
| 睡眠报告页 | report | 报告查看率、分享率 |
| 家长课堂 | course | 内容点击率、完成率 |
| 会员页 | vip | 转化率 |

### 7.4 埋点数据存储

| 类型 | 存储方式 | 保留时间 |
|------|---------|---------|
| 用户行为日志 | 云数据库 | 90天 |
| 聚合统计数据 | 分析平台 | 长期 |
| 原始日志 | 日志服务 | 30天 |
